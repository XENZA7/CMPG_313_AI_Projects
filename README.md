# CMPG313 AI Object Detection

## Overview
This project uses YOLOv8 for detecting and tracking objects in images and videos.  
It demonstrates how AI can identify objects, draw bounding boxes, and save results.

## Features
- Detects objects in single images
- Tracks objects across video frames
- Saves outputs in the `runs/detect/` folder

## Usage
Run:
    py lab_1.py

The script will load the YOLO model, detect objects, display bounding boxes, and save results.

## Example
- Image: 9 persons detected in Asian market.jpg
- Video: Cars, trucks, buses, trains, and persons tracked across frames
